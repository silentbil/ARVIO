# ARVIO 1.9.91

## Key changes
- Improved IPTV categories, context actions, mobile fullscreen TV playback, and IPTV VOD quality selection. Contributors: ARVIO maintainers.
- Fixed Trakt watchlist ordering/matching, watchlist empty-state regressions, and faster Continue Watching startup. Contributors: ARVIO maintainers.
- Fixed watchlist topbar down-navigation so focus returns to the grid correctly. Contributor: Sage Gavin Davids; MR submitted by Eier Kop.
- Improved source switching, playback controls, stream startup, trailer handling, and loading clearlogo polish. Contributors: Sage Gavin Davids, Himanth Reddy, and ARVIO maintainers.
- Added catalog list discovery for Trakt/MDBList and improved catalog focus/navigation stability. Contributors: ARVIO maintainers, Himanth Reddy, and silentbil.
- Improved app language coverage, subtitle language filtering, profile loading, DNS persistence, and cloud login/startup language restore. Contributors: silentbil, Himanth Reddy, Sage Gavin Davids, and ARVIO maintainers.
- Fixed anime episode source matching, details metadata behavior, Crunchyroll assets, and several layout/focus regressions. Contributors: ARVIO maintainers, Sage Gavin Davids, and Himanth Reddy.
